## 手册结构一览

### Fish AI Handbook 内容架构

本手册按照以下结构组织内容：

**主要章节：**

1. **鱼说必看** - 建立AI基础认知
   - Models - 认识主流AI模型
   - 模型排行 - 各模型性能对比
   - 术语扫盲 - 理解关键概念
   - 流行词汇 - AI圈黑话解读

2. **基础用法** - 快速上手AI工具
   - WebChat - 网页聊天界面
   - IDE集成 - 开发环境集成
   - 移动端 - 手机端使用

3. **提示词** - 掌握与AI对话技巧
   - 基础交互 - 入门级对话方法
   - 进阶技巧 - 高级提示词技术
   - 实用范例 - 场景化示例

4. **进阶玩法** - 构建AI应用
   - 知识库/RAG - 检索增强生成
   - MCP - 模型连接协议
   - Agents - AI智能体
   - Workflow - 工作流设计

5. **好玩的** - 探索AI创意应用

6. **资源合集** - 工具与服务推荐

7. **配置指南** - 环境搭建指南
