# 为什么先做环境准备

- **稳定连通**：没有稳定的网络与代理，任何体验都不可靠。
- **更高效率**：顺手的终端、编辑器与包管理器，能把“会用”变成“用得快”。
- **降低门槛**：标准化环境方便你和朋友一起协作、复现问题。

## 你将准备什么

- Windows Terminal（或 iTerm/Terminal）：统一配置的命令行环境
- Visual Studio Code：常用扩展与快捷键
- Node.js：脚手架、前端工具链与 SDK 生态
- GitHub：代码托管与协作
- VPN/代理：连通国外模型与依赖下载
