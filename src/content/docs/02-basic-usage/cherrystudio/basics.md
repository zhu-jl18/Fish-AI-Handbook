Cherry Studio 是一个跨平台的桌面 AI 客户端，支持 Windows、macOS 和 Linux。它的设计理念很简单：把各种 LLM 服务商的模型聚合到一个界面里，让你不用在不同的网页和应用之间来回切换。这不是什么革命性的想法，市面上一堆类似的工具，但至少 Cherry Studio 没把事情搞得一团糟。

## 界面布局

软件的主界面分为三个区域。左侧是侧边栏，用来管理对话历史、助手和各种功能入口。中间是对话区域，显示你和模型的交互内容。右侧是设置面板，可以调整当前对话的参数。这种三栏布局已经被用烂了，但确实没什么好抱怨的，至少比那些把所有东西塞进一个窗口的设计强。

## 对话功能

对话功能是最基础的部分。你在输入框里打字，按回车发送，模型返回回复。如果需要输入多行文本，用 Shift + Enter 换行。这些都是标准操作，如果连这个都要学习，那你可能不适合用电脑。Cherry Studio 支持 Markdown 渲染，所以模型返回的格式化文本、代码块、表格都能正常显示。代码块还带语法高亮，可以一键复制。

模型切换在输入框上方。Cherry Studio 支持的服务商包括 OpenAI、Anthropic、Google Gemini、Azure OpenAI 等主流平台，也支持本地模型服务如 Ollama 和 LM Studio。你可以在对话过程中随时切换模型，不需要重新开始对话。这个设计的好处是可以用不同的模型处理同一个问题，对比它们的输出质量。当然，前提是你得有多个 API 密钥，不然就是在浪费时间。

一个值得一提的功能是"一问多答"。你可以选择多个模型同时回答同一个问题，它们的回复会并排显示在对话区域。这对于评估模型性能或者需要多个角度的答案时很有用。不过这个功能会消耗更多的 API 调用额度，如果你的钱包不够厚，还是省着点用。

## 对话管理

对话历史会自动保存，按助手分组显示在侧边栏。每个对话都有标题，可以手动修改。如果对话太多，可以用搜索功能快速定位。删除对话也很简单，右键点击选择删除即可。Cherry Studio 支持将对话导出为 Markdown、Word 等格式，方便存档或分享。

## 助手系统

助手系统是 Cherry Studio 的另一个核心功能。助手本质上是预设的系统提示词模板。软件内置了 300 多个助手，涵盖编程、写作、翻译、数据分析等场景。这些内置助手的质量参差不齐，有些确实有用，有些就是凑数的。你也可以创建自定义助手，设置名称、描述、系统提示词和默认模型。助手的作用是让你不用每次都重复输入相同的上下文信息，这对于懒人来说是个好功能。

## 文件处理

文件上传功能支持文本、图片、Office 文档、PDF 等格式。上传的文件会被解析成文本内容，作为对话的上下文。这对于需要处理文档内容的场景很实用。不过文件解析的质量取决于文件格式和内容复杂度，有时候可能需要手动调整。如果你的 PDF 是扫描版或者格式很奇葩，别指望软件能完美解析，那不是 Cherry Studio 的问题，是你的文件有问题。

## 个性化设置

Cherry Studio 的设置项比较丰富。你可以调整界面主题、字体大小、代码高亮样式等。快捷键也可以自定义，包括新建对话、打开设置、发送消息等常用操作。如果你习惯用键盘操作，这些快捷键能提高不少效率。

## 数据存储

数据存储方面，Cherry Studio 默认将所有数据保存在本地。对话记录、助手配置、设置项都存储在用户目录下的 `CherryStudio` 文件夹。软件支持 WebDAV 备份，可以将数据同步到支持 WebDAV 协议的云存储服务。这样在多台设备之间切换时，数据可以保持一致。

## API 配置

API 配置是使用 Cherry Studio 的前提。你需要在设置中填入各个服务商的 API 密钥和端点地址。Cherry Studio 支持多密钥轮询，可以用英文逗号分隔多个密钥，软件会自动循环使用。这对于避免单个密钥的速率限制很有帮助。API 密钥会加密存储在本地，不会上传到任何服务器。当然，如果你不信任这个说法，可以自己去看源代码，反正是开源的。

如果你需要使用代理访问某些服务商，可以在设置中配置代理地址。Cherry Studio 支持 HTTP 和 SOCKS5 代理。配置好代理后，所有的 API 请求都会通过代理发送。这个功能对于某些地区的用户来说是必需的，你懂的。

总的来说，Cherry Studio 的基础功能覆盖了日常使用 LLM 的大部分场景。它不是一个复杂的工具，但该有的功能都有，而且实现得比较稳定。如果你需要一个本地的 AI 客户端，而不想在浏览器里打开一堆标签页，Cherry Studio 是个不错的选择。当然，如果你喜欢折腾，市面上还有一堆其他选择，但大多数都是换汤不换药。
