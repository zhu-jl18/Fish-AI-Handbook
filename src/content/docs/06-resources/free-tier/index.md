## Official Channels (Recomended)

这部分是我强烈推荐去白嫖使用的，官方渠道你只用考虑数据被官方卖了，并且稳定性也比乱七八糟的渠道好太多了。

**唯一真神：谷歌**，大善人谷歌有好多白嫖渠道:
- [AI Studio API Key](https://aistudio.google.com/)
- [Build](https://aistudio.google.com/app)
- [Gemini Cli](https://github.com/google-gemini/gemini-cli)

三者分别可以直接使用api-key，通过 build2api 转为 api 进行调用和通过 gcli2api 转为 api，虽然谷歌经常429，不过开两个小号后个日常使用和玩酒馆真的够用了。


很不错的api渠道，可以免费使用国产最新的（开源）大模型：
- [心流 iflow](https://iflow.cn/)
- [魔搭 ModelScope](https://modelscope.cn/)




一些不错的渠道，用来跑翻译或者娱乐，通常是量化降智的，结合号池使用：
- [Cerebras](https://www.cerebras.com/)
- [Nvidia](https://www.nvidia.com/)
- [Groq](https://www.groq.com/)
- [Ollama](https://docs.ollama.com/cloud)
- [无问芯穹](https://cloud.infini-ai.com)
- [Anannas](https://docs.anannas.ai/API/Overview)

比较特殊的，会有新模型上线提供免费测试：
- [OpenRouter](https://openrouter.ai/)

## 公益站点

曾经的我，收集了二十几个公益站点，到最后只留下了几个不错的公益站点，为什么呢？他们的渠道我都知道了，除了一些备用的如gemini，一些太折腾的如claude，我都没留，更何况还有一堆是2api的，使用体验实在太差了，我的公益站2api渠道也基本下完了。我自用的只剩这几个了：
- [AnyRouter](https://anyrouter.top/)
- [ZoneAPI](https://zoneapi.top/)
