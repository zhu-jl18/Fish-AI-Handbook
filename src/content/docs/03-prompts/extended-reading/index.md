## 高级框架速览

- **CRISPE**：强调 Context、Role、Intent、Style、Process、Examples 的全链路提示词拆解，适合需要精细控制语气与流程的任务。
- **RAIL**：通过 Response、Action、Input、Limitation 明确输出边界，可与 JSON Schema 结合约束非法结果。
- **SCQA**：Situation、Complication、Question、Answer 的金字塔结构，帮助在复杂场景中保持叙事逻辑与焦点。
- **PER（Plan-Execute-Review）**：计划-执行-复核三段式，适用于长任务与代码协作，避免一步到位导致的漂移。
- **Dual Loop Prompting**：先生成自检准则，再产出答案，最后用准则审核；常见于需要高可验证性的研究任务。
