## 关键信息（先读这段）
- 提示词公式：目标 + 约束 + 上下文 + 输出 Schema + 示例；先定结构再填内容。
- 上下文工程：System/Developer/User 分层，输入保持“最小充分”，长对话靠滚动摘要与重新宣言。
- 操作路径：骨架模板 → 典型技巧 → 上下文拼装 → 高频场景模板 → 高阶框架。
- 评价标准：可机判、可复用、少废话；无答案时返回空结果而非瞎编。
