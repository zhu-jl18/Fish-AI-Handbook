## 云服务器代理（clash-for-linux-install）

云服务器上跑代理，直接用 clash-for-linux-install，一键装好 mihomo（clash 继任者）。需要 root 或 sudo 权限，shell 用 bash / zsh / fish。

```bash
git clone --branch master --depth 1 https://gh-proxy.com/https://github.com/nelvko/clash-for-linux-install.git \
  && cd clash-for-linux-install \
  && sudo bash install.sh
```

安装时按提示填你的订阅链接（没有就先去搞机场），其他高级用法直接看仓库文档：

- 仓库地址：https://github.com/nelvko/clash-for-linux-install
