玩 AI 必须科学上网。OpenAI、Anthropic、Google 的 API 都需要代理访问。

## 基础概念

| 术语 | 说明 |
| --- | --- |
| **代理 (Proxy)** | 中转服务器，帮你访问目标网站 |
| **VPN** | 虚拟专用网络，加密整个网络流量 |
| **机场** | 付费代理服务商，提供节点订阅 |
| **节点** | 代理服务器，有不同协议（SS/VMess/Trojan等） |
| **订阅链接** | 包含多个节点信息的 URL，客户端自动更新 |

## 推荐配置

**新手**：买机场订阅 + Clash Verge Rev  
**进阶**：自建节点 + V2RayN  
**极客**：链式代理 + 分流规则深度定制

## 注意事项

- 不用免费 VPN 登录重要账号
- 机场选择稳定的，不要贪便宜
- 自建节点注意安全性，定期更新
